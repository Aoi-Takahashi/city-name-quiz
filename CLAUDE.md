## Project Overview

`city-name-quiz` は高知県高知市内の町名と、その町名が属する「大街」を答える町名テストWebアプリ。

## 用語定義

- **町名**：高知市内に存在する町名
- **大街**：町名が属する街のエリア

## 機能要件

- 町名一覧からランダムに20件の町名を抽出して出題する
- 利用者は入力フォームから「町名の読み方」と「大街」を解答する
- 大街の入力はプルダウン形式
- 正解時：正解エフェクトを表示
- 不正解時：不正解箇所の下部に正解を表示し、不正解エフェクトを表示

出題形式例：
```
問題. 以下の町名の読み方と町名が属する大街を解答しなさい
　唐人町　<町名入力欄>　<大街入力プルダウン>
```
